# Coaching-Classes
Developed an informative coaching classes website focused on simplifying the admission process by providing clear details about courses, eligibility, and contact information.

## 🚀 Features
- Displays detailed information about available courses and programs
- Provides admission guidelines and eligibility criteria
- Shows class schedules and timings
- Includes faculty information and contact details
- Simple and user-friendly navigation
- Responsive design for mobile and desktop users

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📂 Installation
1. Clone the repository
2. Open the project folder
3. Run `index.html` in your browser
